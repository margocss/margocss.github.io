Quick Form

<form class="form-group grid-container">

Create a quick form with <form class="form-group">.

For each form-field ( e.g. Username, Password, Email, First Name, Last Name, Occupation, etc. ) in a form-group, you should have a div of <div class="form-field"> that contains the field's<label> & <input>.

Note, a <div class="form-field"> is an inline-block element, so you can force a break to a new line by inserting a <br/> after. 


<form class="form-group">
   
   <div class="form-field">
       <label> Username</label>
       <input type="text"/>
   </div>

   <div class="form-field">
       <label>Password </label>
       <input type="password"/>
   </div>

   <br/>

   <div class="form-field">
       <input type="submit" class="btn" value="Login"/ >
   </div>
  
 </form>

Long Form

<form class="form-group grid-container">

You can also add a class of grid-container to the form to allow for more precise positioning of the form fieldseach <div class="form-field">. Each div with a class of form-field will also include class names describing it's column layout: <div class="form-field sm-12-x-12 md-4-x-12"> 

<form class="form-group grid-container">
   
   <div class="form-field sm-12-x-12 md-4-x-12">
       <label> First </label>
       <input type="text"/>
   </div>

   <div class="form-field sm-12-x-12 md-4-x-12">
       <label>Middle </label>
       <input type="text"/>
   </div>

   <div class="form-field sm-12-x-12 md-4-x-12">
       <label>Last </label>
       <input type="text"/>
   </div>

   <div class="form-field sm-12-x-12 md-6-x-12">
       <label >Email </label>
       <input type="email"/>
   </div>

   <div class="form-field sm-12-x-12 md-6-x-12">
       <label>State </label>
       <select>
         <option value="CA">California </option>
         <option value="NY">New York </option>
         <option value="TX">Texas </option>
     </select>
   </div>

   <div class="sm-12-x-12 md-12-x-12 form-field">
      <label">Description </label>
      <textarea rows="6"> </textarea>
   </div>
  
   <div class="sm-6-x-12 md-6-x-12 form-field">
       <input type="submit" class="btn md primary" value="submit" / >
   </div>
  
 </form>
