About

Goal

The goal of this project is to setup a basic jekyll project scaffold that can be cloned from github that has bootstrap loaded and some sane scss defaults. The idea is to give some structure and code examples for getting a jekyll project off the ground while still promoting and encouraging flexibility in modifying the layout.

How to Use

  1. Clone the repository
  2. npm install will install the dependencies and cli-tools
  3. gulp dev will run the watcher: the sass-compiler and static site generator
