The grid system is responsive and based on a 12 column layout. In order for the grid to work you must have both an outer-div with a class of grid-container and a nested, inner-div with the classes in the format shown above: [screen_sz]-[cols_of_12]-x-12.

You can target column sizes for three screen-widths:

    sm < 750px,
    md = 750px - 1000px
    lg > 1000px.

There are two (2) kinds of grids: default and guttered. The guttered grid has an inner margin of 3% and it is created by adding the class of guttered to a parent-div grid container.
(e.g. <div class="grid-container guttered">...)

Text Formats

All text sizes and line heights are calculated based on ems. The standard paragraph text is sized at 1em. And the default line-height is set to 1.25em on mobile to accomodate the lesser screen height and 1.625em on tablet/desktop for ideal legibility.
