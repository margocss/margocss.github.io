Small CSS Library For Robust Designs

The aim of this project is to provide a set of solid defaults and to give useful HTML patterns for fast prototyping of responsive web sites. Get off the ground quickly using a simple, shallow html tag structure.

Fully Responsive

Most web browsing takes place on mobile devices. Margo was built with responsiveness in mind with typography settings that scale appropriately depending on browser width and easy-to-implement layouts on a 12-column grid.

Neutral base styles and component parts are opt-in

Default styles are neutral and generic, allowing for easy inclusion in a project. Take what you want and leave out what you don't need.

Start simple, then make it more complex

Many components, such as navbars, forms and tables, have simple implementations to get off the ground and then more complex configurations for when your UI becomes more sophisticated.
